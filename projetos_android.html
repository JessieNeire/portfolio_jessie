Jogo Descubra o número

"Descubra o Número" é um jogo interativo projetado para testar a capacidade de adivinhação e a estratégia dos jogadores. Este jogo permite que dois jogadores insiram seus nomes e se enfrentem em uma competição amigável para descobrir um número secreto. O objetivo do jogo é adivinhar corretamente o número secreto, com dicas de "maior" ou "menor" ajudando os jogadores a se aproximarem da resposta correta

Tecnologias utilizadas:
  • Linguagens de Programação: Blocos visuais
  • Ferramentas: Kodular e Kodular Companion

Jogo da Velha

O "Jogo da Velha" é uma versão digital do clássico jogo de estratégia, desenvolvido na plataforma Kodular. Este jogo foi criado para oferecer uma experiência divertida e desafiadora para dois jogadores, com a adição de um placar para acompanhar a pontuação. O objetivo do jogo é alinhar três de seus marcadores (X ou O) em uma linha reta, seja horizontal, vertical ou diagonal, antes do adversário.

Tecnologias utilizadas:
  • Linguagens de Programação: Blocos visuais
  • Ferramentas: Kodular e Kodular Companion

Cadastro Usuário

Este projeto é um sistema de cadastro de usuários, desenvolvido para armazenar e gerenciar informações de usuários utilizando um banco de dados SQLite. O sistema oferece uma interface amigável para registrar, atualizar e excluir dados dos usuários.

Tecnologias utilizadas:
  • Linguagens de Programação: Blocos visuais
  • Banco de Dados: SQLite
  • Ferramentas: Kodular e Kodular Companion

Pesquisar Endereço

Este aplicativo de pesquisa de endereço, permite que os usuários obtenham informações detalhadas de endereços a partir de um CEP específico. Utilizando uma API para a consulta, o aplicativo oferece uma solução prática e rápida para encontrar endereços completos.O aplicativo é facilita a obtenção de informações precisas de endereços por meio de consultas ao CEP, eliminando a necessidade de buscas manuais demoradas e garantindo dados atualizados e confiáveis.

Tecnologias utilizadas:
  • Linguagens de Programação: Blocos visuais
  • API: ViaCEP
  • Ferramentas: Kodular e Kodular Companion
