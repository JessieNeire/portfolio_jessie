Conpac

Conpac é um sistema web para gerenciamento de entregas para condomínios. A portaria recebe a entrega do morador, cadastra no sistema e automaticamente é enviado uma mensagem para o morador via WhatsApp, avisando que sua entrega chegou. O sistema é acesso por meio de login e senha, e o morador pode acessa e verificar seu pacote onde quer que esteja, sem a necessidade de esperar chegar em casa para saber se sua tão esperada entrega chegou.

Tecnologias utilizadas:
  • Linguagens de Programação: HTML, CSS, JS, PHP, SQL
  • Frameworks: W3CSS, jQuery
  • Banco de Dados: MySQL
  • APIs: WhatsApp API
  • Ferramentas: VSCode, Git, GitHub Desktop, phpmyadmin

Curso de DS

Este projeto consiste no desenvolvimento de uma página web para o curso de Técnico em Desenvolvimento de Sistemas na instituição ETEC. O objetivo do projeto foi fornecer uma plataforma informativa e acessível para futuros alunos e demais interessados no curso, destacando informações importantes como o corpo docente, a matriz curricular, os dados de contato, entre outros.

Tecnologias utilizadas:
  • Linguagens de Programação: HTML, CSS, JavaScript
  • Frameworks: W3CSS
  • Ferramentas: Visual Studio Code, Git, GitHub Desktop
